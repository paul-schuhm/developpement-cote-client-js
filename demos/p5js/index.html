Utiliser p5.js dans sa page web

Instructions

  1. Télécharger p5.min.js (le code source Javascript de la bibliothèque p5.js)
  2. Charger le script p5.min.js dans votre page dans une balise script en indiquant la source avec l'attribut src (voir le code source de cette page)
  3. Charger votre propre script sketch.js dans lequel vous déclarez les fonctions setup() et draw()
  4. Écrivez votre code dans le script sketch.js
  5. Have fun !
